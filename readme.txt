Fun default avatars for your WordPress site.

== Description ==

This WordPress plugin will generate a default avatar for each user. Each default avatar is likely to be unique.

== Requirements ==

You will need to compile PHP with the GD library for this plugin to work.

== Frequently Asked Questions ==

= Where are the images stored? =

The images are generated by the plugin and a copy is stored in the uploads folder.

= What is the approximate file size of each generated image? =

3 KB.

= Why are the images blank? =

Blank images are displayed when something is wrong. For example a blank image will be displayed if the GD library isn't available.

= Why are the images broken? =

Broken images are displayed if the images in your uploads folder can't be accessed. This may happen if your server is behind a firewall.

= What default avatar types are available? =

- Pixicon
- Leticon

= How can I change the default avatar type? =

Go to "Settings" and then "Discussion" and then select an option under the "Default Avatar" section.

= Where should I submit bug reports? =

Please open a [GitHub](https://github.com/toucah/default-avatars) issue.
